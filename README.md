- Simple resource pack that translates Minecraft's End Poem into Turkish. Supports all Minecraft versions.

# Installation
1) Download the ```.zip``` file compatible with your version of Minecraft.
2) Open Minecraft, the go to **Settings > Resource Packs**
3) Drag the zip file and choose the pack.
4) Enjoy! <3

*PS: You can use mod loader to download this resource pack too.*
